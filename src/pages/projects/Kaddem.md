# Kaddem Spring boot angular web application

A Web application to manage universities and students. It is a spring boot application with angular front end.

## Technologies

Here are the technologies used in this project:

- Spring boot
- Angular
- Spring data JPA
- Spring security
- Spring web
- Lombok
- Swagger
- MySQL
- Docker
- Sass
- Bootstrap
- Material design
